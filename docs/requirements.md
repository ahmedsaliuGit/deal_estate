# Property Management Dashboard

## Project overview

A web-based property management platform that allows property managers to manages properties, units, leases, rent payment and maintenance requests.

### Users:
**Admin**
Can:
•	Manage all properties
•	Manage users
•	View reports
•	Manage tenants
•	Manage payments

**Property Manager**
Can:
•	Manage assigned properties
•	Manage tenant
•	Records payment
•	Handle maintenance request

**Tenant**
Can:
•	View list information
•	Submit maintenance requests
•	View payment history

### Functional requirements

Authentication
•	User registration
•	User Login
•	Forgot password
•	Reset password
•	Role-base access control

Property management
•	Create property
•	View property
•	Edit property
•	Delete property

Unit management
•	Create unit
•	Assign tenant
•	Mark occupied/vacant

Tenant management
•	Add tenant
•	Edit tenant
•	Remove tenant

Payment management
•	Record payment
•	Generate invoice
•	View payment history

Maintenance 
•	Submit request
•	Assigned artisan
•	Update status

### Non-Functional Requirements
•	Responsive design
•	Secure authentication
•	Fast API responses
•	Mobile friendly UI
